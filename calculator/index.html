Latex Guidelines

Here are some basic LaTeX guidelines for algebra:

  • To write fractions: \frac{numerator}{denominator} (e.g., \frac{a}{b})
  • To write exponents: base^{exponent} (e.g., x^{2})
  • To write subscripts: symbol_{subscript} (e.g., a_{i})
  • To write square roots: \sqrt{expression} (e.g., \sqrt{x+1})
  • To write summations: \sum_{i=1}^{n} expression (e.g., \sum_{i=1}^{n} i)
  • To write products: \prod_{i=1}^{n} expression (e.g., \prod_{i=1}^{n} i)
  • To write integrals: \int_{lower}^{upper} expression \, dvariable (e.g., \int_{0}^{1} x^2 \, dx)
  • To write greek letters: \alpha, \beta, \gamma, \pi, \theta (e.g., \pi, \theta)
